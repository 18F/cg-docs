Estimate your monthly cost

Monthly costs are based on the package fee (i.e., Prototyping, FISMA Low, FISMA Moderate) plus memory usage. Usage costs are based on the amount of memory (RAM) that you reserve for your system ($105/GB RAM per month).

Most cloud.gov customers typically reserve 2–8 GB of memory to cover multiple applications across development, staging, and production environments.

Example

Suppose you have 2 apps that each need 512 MB (0.5 GB) memory. Each app needs 1 instance in dev, 1 instance in staging, and 2 in production. The apps require data — nothing sensitive — so a FISMA Low production package will be adequate for hosting.

Access Fee

FISMA Low $1667

Memory Usage Cost

[ (2 x 0.5 GB) + (4 x 0.5 GB) + (2 x 0.5 GB) ] x $105 $420

Total monthly cost $2087
